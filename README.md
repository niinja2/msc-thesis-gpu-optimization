Summary

This thesis investigates CPU and GPU acceleration of N-body simulations with a focus on performance optimization and hardware utilization. The work explores the relationship between algorithm design and hardware characteristics through benchmarking and performance analysis.

Topics explored include:

• CPU and GPU parallelization
• Memory hierarchy
• Cache locality
• Memory latency
• Memory bandwidth limitations
• Floating-point throughput
• Memory blocking
• Loop unrolling
• False sharing
• Performance scaling
• Heterogeneous computing
• Workload balancing
• Asynchronous execution

Relevance Today

Although completed in 2016 using Java, Aparapi and GPU technologies available at the time, many of the underlying concepts remain relevant to modern high-performance computing and AI systems, including memory bottlenecks, data movement, memory access patterns, bandwidth utilization, hardware utilization and performance optimization.
